Import your notification details

Use one of these. Then new help requests will notify you by Gmail, Telegram, SMS, or WhatsApp (all free).

Option A – File (local or deploy with repo)

1. Copy notifications.example.json and rename to notifications.json in the same folder.
2. Fill in your email, Telegram, SMS, WhatsApp details in that file.
3. For Vercel: add notifications.json to the repo and deploy (or use Option B).

Option B – Vercel Environment Variables

Vercel → your project → Settings → Environment Variables. Add these names and your values:

Gmail:
SOULPLACE_SMTP_HOST   SOULPLACE_SMTP_PORT   SOULPLACE_SMTP_USER   SOULPLACE_SMTP_PASSWORD
SOULPLACE_EMAIL_FROM   SOULPLACE_EMAIL_TO   SOULPLACE_SMTP_TLS
Telegram:
TELEGRAM_BOT_TOKEN   TELEGRAM_CHAT_ID
SMS:
SOULPLACE_SMS_TO
WhatsApp (use one):
CallMeBot: CALLMEBOT_WHATSAPP_PHONE   CALLMEBOT_WHATSAPP_APIKEY
WA Bridge: SOULPLACE_WHATSAPP_TO   WABRIDGE_API_KEY   wabridge.cloud
Green API: SOULPLACE_WHATSAPP_TO   GREEN_API_ID_INSTANCE   GREEN_API_TOKEN   console.green-api.com

Only set the channels you want. See NOTIFICATIONS.txt in the project for how to get Telegram token, Gmail app password, CallMeBot key, etc.
