Import notification details

Use one of these so new help requests can notify you (Gmail, Telegram, SMS, WhatsApp).

Option A – File

Copy notifications.example.json to notifications.json, fill in your details. For Vercel: add the file to the repo and deploy.

Option B – Vercel Environment Variables

Vercel → Project → Settings → Environment Variables. Add: SOULPLACE_SMTP_*, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID, SOULPLACE_SMS_TO, GREEN_API_* (see NOTIFICATIONS.txt).
