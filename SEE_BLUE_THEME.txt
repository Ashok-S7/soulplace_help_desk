The app is already using the BLUE theme in the code. If you still see orange:

  1. HARD REFRESH (clear cache)
     • Chrome / Edge: Press Ctrl + Shift + R  (or Ctrl + F5)
     • Or: Right‑click Refresh button → "Empty Cache and Hard Reload"

  2. If you use VERCEL – deploy the latest code
     • Push your project to GitHub, then Vercel will redeploy.
     • Or in PowerShell:
       cd "c:\mini project\soulplace_help_desk"
       git add -A
       git commit -m "Blue theme and cache bust"
       git push

  3. After that, open the login/dashboard again. Buttons and links will be BLUE.

The CSS file now has ?v=2 so the browser will load the new styles after refresh or redeploy.
