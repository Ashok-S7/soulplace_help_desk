    or no production deployment). The app code is fine on kappa.
  • Fix in Vercel (pick one):
    (A) Use the kappa URL above everywhere; or
    (B) Open project "soulplace-help-desk" → Settings → Git → confirm repo;
        Root Directory = soulplace_help_desk; Redeploy Production; or
    (C) Project kappa → Settings → Domains → add soulplace-help-desk.vercel.app
        and remove it from the broken project so traffic hits the live deployment.
