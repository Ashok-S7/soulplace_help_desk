Notification sound – Maroon 5 "Animals"

Place your notification sound file here with this exact name:

  maroon_5_animals.mp3

Copy from your Downloads (or anywhere) to this folder, e.g.:
  From: C:\Users\acer\Downloads\maroon_5_animals.mp3
  To:   soulplace_help_desk\static\sounds\maroon_5_animals.mp3

Full path on your PC (example):
  C:\mini project\soulplace_help_desk\static\sounds\maroon_5_animals.mp3

The dashboard will play this file when a new help request arrives.
If the file is missing, a short beep is used instead.

Supported formats: MP3 is recommended (works in all browsers).

Take access: Staff and admin – on PC or phone, tap/click "Enable notifications"
or "Allow notification sound" once. The app will request access for: (1) in-app popup + Maroon sound;
(2) if you enable notifications, lock-screen notifications with Maroon sound too.
